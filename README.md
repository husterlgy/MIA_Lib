# MIA_Lib 
 成员推断攻击库 Membership Inference Attack Library
## 本库所复现的成员推断攻击

 - **Shadow Attack**：Membership Inference Attacks Against Machine Learning Models
 - **ML-Leaks：** ML-Leaks: Model and Data Independent Membership Inference Attacks and Defenses on Machine Learning Models
 - **Grad Attack：** Comprehensive Privacy Analysis of Deep Learning 
 - **Output Attack：** Comprehensive Privacy Analysis of Deep Learning 中仅使用模型最后一层的攻击
